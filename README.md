# ⏱ Task 02 – Advanced Stopwatch Web Application  
Web Development Internship – SkillCraft Technology  

---

## 📌 Project Overview  

This project was developed as part of **Task 02** of the Web Development Internship at **SkillCraft Technology**.

The objective was to create an interactive, dynamic, and user-friendly stopwatch web application with start, pause, reset, and lap tracking functionality.

The application is designed with a modern, professional UI and includes advanced features such as millisecond precision, lap analytics, and keyboard shortcuts.

## ✨ Features  

- Start / Pause toggle functionality  
- Reset stopwatch  
- Millisecond precision timing  
- Lap time tracking  
- Fastest lap highlighting  
- Slowest lap highlighting  
- Dynamic lap numbering  
- Keyboard shortcuts support  
- Visual running status indicator  
- Fully responsive design  
- Modern glassmorphism UI  
- Animated gradient background  

---

## ⌨ Keyboard Controls  

| Key | Function |
|------|----------|
| Space | Start / Pause |
| L | Add Lap |
| R | Reset |
